# lynx: resumeExposure() static method

<RuntimeBadge type={'bts'} />

`lynx.resumeExposure` 方法可以被用来开启曝光检测，即重新开始检测目标节点的可见性，后续正常触发曝光/反曝光事件。

## 语法

```ts
resumeExposure(): void;
```

### 参数

无（`undefined`）。

### 返回值

无（`undefined`）。

## 注意事项

调用 `lynx.resumeExposure` 不一定会触发曝光事件，是否触发曝光事件在于这一次与上一次曝光检测的结果差异，具体参考[曝光能力](../../../guide/interaction/visibility-detection/exposure-ability.mdx)。
