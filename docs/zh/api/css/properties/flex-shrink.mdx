# flex-shrink

## 介绍

定义弹性项目在**主轴**的缩放系数。弹性容器将元素布局完后，如果存在元素溢出，会根据元素的缩放系数比例将对应的元素进行缩放保证元素不会溢出容器。

## 语法

```css
/* <number> 值 */
flex-shrink: 2;
flex-shrink: 0;
```

### 取值

- [`<number>`](api/css/data-type/number.mdx)

  负值无效，`0` 表示不会缩放，**默认值**为 `1`。
