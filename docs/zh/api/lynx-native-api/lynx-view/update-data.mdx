# UpdateData

使用 [LynxUpdateMeta](/api/lynx-native-api/lynx-update-meta) 更新 `LynxView`，是客户端更新模板数据的主入口。

## 语法

### Android

```java
void updateData(LynxUpdateMeta meta)
```

### 参数说明

- `meta: LynxUpdateMeta`: Lynx 模板更新元数据，开发者通过该数据结构指定更新模板的可选数据内容；

### iOS

```objc
(void)updateData:(nonnull LynxUpdateMetadata*)meta;
```

### 参数说明

- `meta: LynxUpdateMeta*`: Lynx 模板更新元数据，开发者通过该数据结构指定更新模板的可选数据内容；
