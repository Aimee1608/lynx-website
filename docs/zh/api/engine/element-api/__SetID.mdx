# `__SetID`

## 介绍

设置 id 选择器给特定的 Element 节点。

## 语法

```jsx
__SetID(element: Element，id: string) : void;
```

### 参数

|   Key   | 说明                                                      |
| :-----: | --------------------------------------------------------- |
| element | 任意的 Element 对象。                                     |
|   id    | 需要设置给 Element 对象的id selector，类型需要为 string。 |

### 返回值

无返回值。

## 示例

前端框架可以把对应的前端标签编译期处理为如下所示 render 指令。

```jsx
<view id="video" />
```

```tsx
// main-thread.js
let element = __CreateElement('view', 0, {});
__SetID(element, 'video');
```
