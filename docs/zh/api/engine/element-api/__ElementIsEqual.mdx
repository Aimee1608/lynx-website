# `__ElementIsEqual`

## 介绍

判断传入的两个 Element 对象是否为同一个 Element。

## 语法

```jsx
__ElementIsEqual(left: Element, right: Element) : bool;
```

### 参数

|  Key  | 说明                  |
| :---: | --------------------- |
| left  | 被比较的 left 节点。  |
| right | 被比较的 right 节点。 |

### 返回值

只有当 left 和 right 为同一个节点时，才返回 true。

## 示例

前端框架可以按下如下的方法进行判断操作

```jsx
// main-thread.js
let childA = __CreateElement('view', 0, {});

let childB = __CreateElement('view', 0, {});

let isEqual = __ElementIsEqual(childA, childB);
```
