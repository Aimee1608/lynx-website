# `__AddDataset`

## 介绍

新增 data- 属性至 Element 对象，不会影响已有的 data- 属性。

## 语法

```jsx
__AddDataset(element: Element, key: string, value: any) : void;
```

### 参数

|   Key   | 说明                                    |
| :-----: | --------------------------------------- |
| element | 任意的 Element 对象。                   |
|   key   | 表示 dataset key 的大小写敏感的字符串。 |
|  value  | 表示 dataset value，可以为任意类型。    |

### 返回值

无返回值。

## 示例

前端框架可以按下如下的方法生成 render 指令。

```jsx
<view src="xxx" data-key="value" />
```

```jsx
// main-thread.js
let element = __CreateElement('view', 0, {});
__AddDataset(element, 'key', 'value');
```
