# `__NextElement`

## 介绍

返回指定 Element 节点在其父节点的子节点列表中紧跟其后的节点，如果指定节点是最后一个节点，则返回 undefined。

## 语法

```jsx
__NextElement(element: Element) : Element|undefined;
```

### 参数

|   Key   | 说明                  |
| :-----: | --------------------- |
| element | 任意的 Element 对象。 |

### 返回值

返回当前 Element 对象在其父节点的子节点列表中紧跟在其后面的节点，如果当前 Element 为最后一个节点，则返回 undefined。

## 示例

前端框架可以按下如下的方法获取指定 Element 节点的后一个节点。

```jsx
// main-thread.js
let element = __CreateElement('view', 0, {});

let next = __NextElement(element);
```
