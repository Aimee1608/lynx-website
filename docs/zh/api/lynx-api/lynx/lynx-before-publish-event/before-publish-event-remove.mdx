# BeforePublishEvent: remove() method

`remove` 方法可以被用来移除对某个 `DOM` 事件的监听。

## 语法

```ts
remove(name: string, listener: () => {}): BeforePublishEvent;
```

### 参数

#### name

```ts
name: string;
```

指定被监听的 `DOM` 事件的名称。

#### listener

```ts
listener: () => {};
```

指定监听回调。

### 返回值

`BeforePublishEvent` 对象。
