# `__GetDataByKey`

## 介绍

返回 Element 特定的 data- 属性的值。

## 语法

```jsx
__GetDataByKey(element: Element, key: string) : any;
```

### 参数

|   Key   | 说明                            |
| :-----: | ------------------------------- |
| element | 任意的 Element 对象。           |
|   key   | data- key，大小写敏感的字符串。 |

### 返回值

当前 Element 的 data- key 所对应的 data- value，可能是任意类型。

## 示例

前端框架可以按下如下的方法获取 Element 的 data- 属性的值。

```jsx
// main-thread.js
let element = __CreateElement('view', 0, {});
__SetDataset(element, { key: 'value' });

let dataset = __GetDataByKey(element, 'key');
```
