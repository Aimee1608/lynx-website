# BeforePublishEvent: add() method

`add` 方法可以被用来注册对某个元件事件的监听。

## 语法

```ts
add(name: string, listener: () => {}, context?: BaseInstance): BeforePublishEvent;
```

### 参数

#### name

```ts
name: string;
```

指定需要监听的元件事件的名称。

#### listener

```ts
listener: () => {};
```

指定监听回调。

#### context

```ts
context?: BaseInstance
```

前端自定义 `component` 或 `card` 实例。

### 返回值

`BeforePublishEvent` 对象。
