# `__GetID`

## 介绍

返回 Element 的 id 选择器。

## 语法

```jsx
__GetID(element: Element) : string;
```

### 参数

|   Key   | 说明                  |
| :-----: | --------------------- |
| element | 任意的 Element 对象。 |

### 返回值

当前 Element 的 id 选择器，若没有设置过，则返回空字符串。

## 示例

前端框架可以按下如下的方法获取 Element id 选择器。

```jsx
// main-thread.js
let element = __CreateElement('view', 0, {});
__SetID(element, 'video');

let id = __GetID(element);
```
