# `__AppendElement`

## 介绍

在当前 Element 的最后一个子节点之后插入一个 Element 对象。

## 语法

```jsx
__AppendElement(parent: Element, child: Element) : Element;
```

### 参数

|  Key   | 说明                        |
| :----: | --------------------------- |
| parent | 父 Element 对象。           |
| child  | 要追加给父节点的child节点。 |

### 返回值

返回追加后的 child 子节点。

## 示例

前端框架可以按下如下的方法进行插入操作

```jsx
// main-thread.js
let child = __CreateElement('view', 0, {});

let parent = __CreateElement('view', 0, {});

__AppendElement(parent, child);
```
