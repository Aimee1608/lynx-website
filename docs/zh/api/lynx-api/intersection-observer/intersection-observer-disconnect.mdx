# IntersectionObserver: disconnect() method

`disconnect` 方法可以被用来清除目标节点与回调，不再观察目标节点，也不会触发相应回调，如果需要重新观察目标节点，需要重新指定目标节点与回调。

## 语法

```ts
disconnect(): void;
```

### 参数

无（`undefined`）。

### 返回值

无（`undefined`）。
