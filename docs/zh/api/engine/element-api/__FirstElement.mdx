# `__FirstElement`

## 介绍

返回 Element 节点的第一个子节点。

## 语法

```jsx
__FirstElement(element: Element) : Element|undefined;
```

### 参数

|   Key   | 说明                  |
| :-----: | --------------------- |
| element | 任意的 Element 对象。 |

### 返回值

返回当前 Element 的第一个子 Element

## 示例

前端框架可以按下如下的方法获取 Element 的第一个子 Element

```jsx
// main-thread.js
let element = __CreateElement('view', 0, {});

let first = __FirstElement(element);
```
