# `__SetInlineStyles`

## 介绍

设置内联样式给特定的 Element 节点。

## 语法

```jsx
__SetInlineStyles(element: Element, value: string|object) : void;
```

### 参数

|   Key   | 说明                                                      |
| :-----: | --------------------------------------------------------- |
| element | 任意的 Element 对象。                                     |
|  value  | 内联样式，覆盖原有的内联样式，类型为 string 或者 object。 |

### 返回值

无返回值。

## 示例

前端框架可以把对应的前端标签编译期处理为如下所示 render 指令。

```jsx
<view src="xxx" style={top:'10px', left: '10px', right: '10px', bottom: '10px'}/>
```

```
// main-thread.js
let element = __CreateElement("view", 0, {});
__SetAttribute(element, "src", "xx");

__SetInlineStyles(element, {top:'10px', left: '10px', right: '10px', bottom: '10px'});
```
