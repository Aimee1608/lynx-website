# `<percentage>`

`<percentage>` 表述一个百分比值。许多 CSS 属性都可以取百分比值，经常用以根据父对象来确定大小。百分比值由一个 [`<number>`](./number.mdx) 具体数值后跟着 `%` 符号构成，在 `%` 和数值之间是不允许有空格。

许多长度属性使用百分比，如 `width`、`height`、`margin`、`padding`。

## 示例

```tsx
<view>
  <text style={{ width: '50%', marginLeft: '20%', backgroundColor: '#00FF00' }}>
    Width: 50%, Left margin: 20%
  </text>
  <text style={{ width: '30%', marginLeft: '60%', backgroundColor: '#FF0000' }}>
    Width: 30%, Left margin: 60%
  </text>
</view>
```
