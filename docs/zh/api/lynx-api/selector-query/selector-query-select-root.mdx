# SelectorQuery: selectRoot() method

选择 `SelectorQuery` 所指定的根节点。

通过 `lynx.createSelectorQuery()` 使用时，选择的元件为整个页面的根元件 `<page>`。

## 语法

```ts
selectRoot(): NodesRef;
```

### 参数

无。

### 返回值

代表该查询结果的 `NodesRef` 对象实例。

## 示例

将整个页面置灰:

```tsx
lynx
  .createSelectorQuery()
  .selectRoot()
  .setNativeProps({
    filter: 'grayscale(100%)',
  })
  .exec();
```
