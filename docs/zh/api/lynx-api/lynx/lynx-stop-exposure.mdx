# lynx: stopExposure() static method

<RuntimeBadge type={'bts'} />

`lynx.stopExposure` 方法可以被用来停止曝光检测，即不再检测目标节点的可见性，后续不会再触发曝光/反曝光事件，可以调用 [lynx.resumeExposure](./lynx-resume-exposure.mdx) 来重新开始曝光检测。

## 语法

```ts
stopExposure(options?: StopExposureOptions): void;
```

### 参数

#### options

```ts
interface StopExposureOptions {
  sendEvent?: boolean;
}
```

指定 `lynx.stopExposure` 的行为：

- `sendEvent`：指定是否发送目前所有可见目标节点的反曝光事件，默认为 `true`。

### 返回值

无（`undefined`）。
