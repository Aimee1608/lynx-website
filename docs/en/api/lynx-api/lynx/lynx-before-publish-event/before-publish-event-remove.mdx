# BeforePublishEvent: remove() method

<RuntimeBadge type={'bts'} />

The `remove` method can be used to remove listening for a certain `DOM` event.

## grammar

```ts
remove(name: string, listener: () => {}): BeforePublishEvent;
```

### Parameters

#### name

```ts
name: string;
```

Specifies the name of the `DOM` event to be listened to.

#### listener

```ts
listener: () => {};
```

Specify the listening callback.

### return value

`BeforePublishEvent` object.
