这里跟 Android 普通视图的排版约束是一致，比如给 LynxView 的设置排版参数是 [WRAP_CONTENT](https://developer.android.com/reference/android/view/ViewGroup.LayoutParams#WRAP_CONTENT)，即可让 LynxView 的大小根据内容自适应:

```java
parentView.addView(lynxview,
    new ViewGroup.LayoutParams(ViewGroup.LayoutParams.WRAP_CONTENT,ViewGroup.LayoutParams.WRAP_CONTENT));
```
