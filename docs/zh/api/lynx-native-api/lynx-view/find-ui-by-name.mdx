# findUIByName

根据 `name` 属性查找 `LynxUI` 节点；

## 语法

### Android

```java
public LynxBaseUI findUIByName(String name);
```

### 参数说明

- `name: String`：前端在布局文件中指定的视图的 name 属性；

### iOS

```objc
- (nullable LynxUI*)uiWithName:(nonnull NSString*)name;
```

### 参数说明

- `name: NSString*`：前端在布局文件中指定的视图的 name 属性；
