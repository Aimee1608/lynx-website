# lynx: cancelAnimationFrame() static method

取消一个 [`requestAnimationFrame`] 的回调。

## 语法

```ts
cancelAnimationFrame: (requestID: number) => void;
```

### 参数

#### requestID

需要取消的 [`requestAnimationFrame`] 的返回值。

### 返回值

无（`undefined`）。

[`requestAnimationFrame`]: ./lynx-request-animation-frame.mdx
