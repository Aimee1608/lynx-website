# clearTimeout() global function

取消由 `setTimeout` 设置的定时器。

## 语法

```typescript
  clearTimeout(timerId: number): void;
```

### 参数

#### timerId

`setTimeout` 返回的定时器 ID。如果该 ID 对应的定时器已经执行完毕，则该方法不会执行任何操作。

**传入一个非 `setTimeout` 返回的 ID 值是未定义行为，可能导致问题。**

## 注意事项

1. `setTimeout()` 和 `setInterval()` 共用一个编号池，因此技术上 `clearTimeout()` 和 `clearInterval()` 可以互换。但是，为了避免混淆，不要混用取消定时函数。
