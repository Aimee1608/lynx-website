# updateFontScale

客户端设置更改字体缩放比例，会自动更改文本的大小；

## 语法

### Android

```java
public void updateFontScale(float scale);
```

### 参数说明

- `scale: float`: 字体缩放比例；

### iOS

```objc
- (void)updateFontScale:(CGFloat)scale;
```

### 参数说明

- `scale: CGFloat`: 字体缩放比例；
