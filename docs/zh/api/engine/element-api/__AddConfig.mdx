# `__AddConfig`

## 介绍

新增 config key 和 config value 至 Element 对象。

## 语法

```jsx
__AddConfig(element: Element, type: string, value: any) : void;
```

### 参数

|   Key   | 说明                                   |
| :-----: | -------------------------------------- |
| element | 任意的 Element 对象。                  |
|  type   | 表示 config key 的大小写敏感的字符串。 |
|  value  | 表示 config value，可以为任意类型。    |

::: tip 备注
Developer Service 开发者给 Element 设置一些自定义 config，方便前端框架进行一些拓展。
:::

### 返回值

无返回值。

## 示例

前端框架可以按下如下的方法生成 render 指令。

```jsx
<view src="xxx" bindtap="onTap" />
```

```tsx
// main-thread.js
let element = __CreateElement('view', 0, {});
__SetAttribute(element, 'src', 'xx');

__AddConfig(element, 'anyConfig', {});
```
