# `__GetParent`

## 介绍

返回 Element 节点的父节点。

## 语法

```jsx
__GetParent(element: Element) : Element|undefined;
```

### 参数

|   Key   | 说明                  |
| :-----: | --------------------- |
| element | 任意的 Element 对象。 |

### 返回值

返回当前节点的父节点，若没有父节点，则返回 undefined。

## 示例

前端框架可以按下如下的方法获取 Element 的父亲节点。

```jsx
// main-thread.js
let element = __CreateElement('view', 0, {});

let parent = __GetParent(element);
```
