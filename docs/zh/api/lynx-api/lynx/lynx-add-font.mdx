# lynx: addFont() static method

<RuntimeBadge type={'bts'} />

用于动态添加字体。

## 语法

```ts
export interface FontFace {
  'font-family': string;
  'src': string;
}

addFont(fontFace: FontFace, callback?: () => void): void;
```

### 参数

#### fontFace

需要添加的字体。

#### callback

添加字体之后的回调。

## 示例

```ts
lynx.addFont(
  {
    'font-family': 'CustomFont',
    src: 'url("xxx")',
  },
  () => {
    console.log('load font');
  },
);
```
