# `<length-percentage>`

相当于 \[[`<length>`]|[`<percentage>`]\]，其中 [`<percentage>`] 将解析为 [`<length>`]。另外支持 `calc()` 数学表达式。

## 语法

一个 `<length-percentage>` 可以是 [`<length>`]、[`<percentage>`] 或 `calc()` 数学表达式。`calc()` 函数是一个数学函数，允许对数值进行基本算术运算，使用加法（+）、减法（-）、乘法（\*）、除法（/）和括号。
