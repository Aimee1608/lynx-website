# clearInterval() global function

取消由 `setInterval` 设置的定时器。

## 语法

```typescript
clearInterval(timerId: number): void;
```

### 参数

#### timerId

`setInterval` 返回的定时器 ID。**传入一个非 `setInterval` 返回的 ID 值是未定义行为，可能导致问题。**

## 注意事项

1. `setTimeout()` 和 `setInterval()` 共用一个编号池，因此技术上 `clearTimeout()` 和 `clearInterval()` 可以互换。但是，为了避免混淆，不要混用取消定时函数。
