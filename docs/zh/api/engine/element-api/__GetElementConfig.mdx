# `__GetElementConfig`

## 介绍

以 map 的形式返回 Element 节点的所有 config。

## 语法

```jsx
__GetElementConfig(element: Element) : Record<string, any>;
```

### 参数

|   Key   | 说明                  |
| :-----: | --------------------- |
| element | 任意的 Element 对象。 |

### 返回值

当前 Element 的 config，返回值为一个 map。

## 示例

前端框架可以按下如下的方法获取 Element 的 config。

```jsx
// main-thread.js
let element = __CreateElement('view', 0, {});
__SetConfig(element, { anyConfig: {} });

let listener = __GetElementConfig(element);
```
