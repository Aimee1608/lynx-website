# `__SetAttribute`

## 介绍

设置 attribute 给特定的 Element 节点。

## 语法

```jsx
__SetAttribute(element: Element, key: string, value: any) : void;
```

### 参数

|   Key   | 说明                                       |
| :-----: | ------------------------------------------ |
| element | 任意的 Element 对象。                      |
|   key   | 类型为 string，需要设置的 attribute key。  |
|  value  | 类型为任意值，需要设置的 attribute value。 |

### 返回值

无返回值。

## 示例

前端框架可以按下如下的方法生成 render 指令。

```jsx
<view src="xxx" />
```

```tsx
// main-thread.js
let element = __CreateElement('view', 0, {});
__SetAttribute(element, 'src', 'xx');
```
