# `__GetChildren`

## 介绍

以 array 的形式按顺序返回 Element 的所有子节点。

## 语法

```jsx
__GetChildren(element: Element) : Array<Element>;
```

### 参数

|   Key   | 说明                  |
| :-----: | --------------------- |
| element | 任意的 Element 对象。 |

### 返回值

以数组的形式返回当前 Element 的所有子节点。

## 示例

前端框架可以按下如下的方法获取 Element 的所有子节点。

```jsx
// main-thread.js
let element = __CreateElement('view', 0, {});

let children = __GetChildren(element);
```
