# lynx: querySelector(selector: string)

在页面的子元素中查找第一个满足指定选择器的元素。返回一个 [`MainThread.Element`](../main-thread/main-thread-element)。如果没有满足要求的元素，返回 `null`。支持的选择器列表见 [selector](../selector-query/selector-query-select#selector)。

## 语法

```ts
const element = lynx.querySelector(selector);
```

### 参数

#### selector

指定选择器，支持的选择器列表见 [selector](../selector-query/selector-query-select#selector)。

### 返回值

[`MainThread.Element`](../main-thread/main-thread-element) 或 `null`。
