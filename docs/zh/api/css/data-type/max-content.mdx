# `<max-content>`

`max-content` 目前支持在 [`width`]、[`height`] 和[网格布局]（grid）属性 ([`grid-template-columns`]、[`grid-template-rows`]、[`grid-auto-columns`]、[`grid-auto-rows`]) 中使用。

## 示例

```css
width: max-content;
grid-template-columns: max-content;
grid-template-rows: max-content;
grid-auto-columns: max-content;
grid-auto-rows: max-content;
```
