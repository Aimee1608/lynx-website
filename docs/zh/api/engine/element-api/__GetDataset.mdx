# `__GetDataset`

## 介绍

以 map 的形式返回 Element 节点的所有 data- 属性。

## 语法

```jsx
__GetDataset(element: Element) : Record<string, any>;
```

### 参数

|   Key   | 说明                  |
| :-----: | --------------------- |
| element | 任意的 Element 对象。 |

### 返回值

当前 Element 的 dataset，返回值为一个 map。

## 示例

前端框架可以按下如下的方法获取 Element 的 dataset。

```jsx
// main-thread.js
let element = __CreateElement('view', 0, {});
__SetDataset(element, { key: 'value' });

let dataset = __GetDataset(element);
```
