# `__RemoveElement`

## 介绍

将 Element 节点从其父节点的子节点列表中删除。

## 语法

```jsx
__RemoveElement(parent: Element, child: Element) : Element;
```

### 参数

|  Key   | 说明                     |
| :----: | ------------------------ |
| parent | 父节点对象。             |
| child  | 从父节点中移除的子节点。 |

### 返回值

返回被移除的 child 子节点。

## 示例

前端框架可以按下如下的方法进行移除操作。

```jsx
// main-thread.js
let child = __CreateElement('view', 0, {});

let parent = __CreateElement('view', 0, {});

__AppendElement(parent, child);

__RemoveElement(parent, child);
```
