# `__SetClasses`

## 介绍

设置类选择器给特定的 Element 节点。

## 语法

```jsx
__SetClasses(element: Element, classNames: string) : void;
```

### 参数

|    Key     | 说明                                                  |
| :--------: | ----------------------------------------------------- |
|  element   | 任意的 Element 对象。                                 |
| classNames | 新的类选择器列表，覆盖原有的类选择器，类型为 string。 |

### 返回值

无返回值。

## 示例

前端框架可以把对应的前端标签编译期处理为如下所示 render 指令。

```jsx
<view src="xxx" class="A B C" />
```

```tsx
// main-thread.js
let element = __CreateElement('view', 0, {});
__SetAttribute(element, 'src', 'xx');
__SetClasses(element, 'A B C');
```
