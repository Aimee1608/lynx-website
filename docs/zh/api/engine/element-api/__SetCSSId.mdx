# `__SetCSSId`

## 介绍

设置 css id 给特定的 Element 节点。

## 语法

```jsx
__SetCSSId(element: Element, number: cssID) : void;
```

### 参数

|   Key   | 说明                         |
| :-----: | ---------------------------- |
| element | 任意的 Element 对象。        |
|  cssID  | css id，是一个 number 类型。 |

### 返回值

无返回值。

## 示例

前端框架可以按下如下的方法进行替换操作。

```jsx
// main-thread.js
let element = __CreateElement('view', 0, {});
__SetCSSId(element, 0);
```
