# `__SetDataset`

## 介绍

设置 data- 属性给特定的 Element 节点。

## 语法

```jsx
__SetDataset(element: Element, dataset: Record<string, any>) : void;
```

### 参数

|   Key   | 说明                                                  |
| :-----: | ----------------------------------------------------- |
| element | 任意的 Element 对象。                                 |
| dataset | 需要设置给 Element 对象的 dataset ，是一个 map 对象。 |

### 返回值

无返回值。

## 示例

前端框架可以按下如下的方法生成 render 指令。

```jsx
<view src="xxx" data-key="value" />;

// main-thread.js
let element = __CreateElement('view', 0, {});
__SetDataset(element, { key: 'value' });
```
