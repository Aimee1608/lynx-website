# `__GetTag`

## 介绍

返回 Element 节点的标签选择器。

## 语法

```jsx
__GetTag(element: Element) : string;
```

### 参数

|   Key   | 说明                  |
| :-----: | --------------------- |
| element | 任意的 Element 对象。 |

### 返回值

返回值是 Element 对象的标签选择器，类型为 string。

## 示例

前端框架可以按下如下的方法获取 Element 的标签选择器。

```jsx
let element = __CreateElement('view', 0, {});
let tag = __GetTag(element);
```
