# `__CreatePage`

## 介绍

创建 PageElement 对象。

## 语法

```jsx
__CreatePage(componentID: string, cssID: number, info: Record<string, any>|null|undefined) : Element;
```

### 参数

|     Key     | 说明                                                                                                                                                                     |
| :---------: | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| componentID | 前端框架设置给 PageElement 的 componentID，是大小写敏感的字符串。PageElement 是一个特殊的 ComponentElement，后续在介绍ComponentElement 时会详细介绍 componentID 的作用。 |
|    cssID    | CSSFragment id，是一个 number 类型。                                                                                                                                     |
|    info     | PageElement 创建时所需的一些额外信息，该参数可以省略。                                                                                                                   |

### 返回值

返回 PageElement。

## 示例

前端框架可以按下如下的方法创建

```jsx
// main-thread.js
let page = __CreatePage('0', 0, {});
```
