# `__GetComponentID`

## 介绍

返回 ComponentElement 的 componentID。

## 语法

```jsx
__GetComponentID(element: Element) : string;
```

### 参数

|   Key   | 说明                      |
| :-----: | ------------------------- |
| element | 任意的 ComponentElement。 |

### 返回值

返回 ComponentElement 的 componentID。

## 示例

前端框架可以按下如下的方法获取 ComponentElement 的 componentID。

```jsx
// main-thread.js
let element = __CreateComponent(0, '1', 1, '', 'name', 'path', {}, {});

let id = __GetComponentID(element);
```
